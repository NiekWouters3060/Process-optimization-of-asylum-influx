# Process-optimization-of-asylum-influx
Developing a data-driven framework to manage and predict the pressure on CGVS (Belgian asylum agency). The focus is on predicting the monthly inflow and simulating more efficient processing processes through intelligent prioritization. All the used data is public data from the Belgian government published on https://www.cgvs.be/nl/actueel.

Deel 1: Collecting en Cleaning Data, EDA en PowerBI interactief dashboard dat data per nationaliteit weergeeft.

Deel 2: Ontwikkelde een model dat de maandelijkse instroom voorspelt met een [X]% nauwkeurigheid (MAE), waardoor een proactieve schatting van de benodigde verwerkingscapaciteit mogelijk wordt.

Deel 3: Als tweede luik van het project bouwde ik, op de gedachtengang dat dit mogelijks een interessante praktijktoepassing zou zijn, een logistic regression model of decision tree te bouwen gebasseerd op data van individuele cases en vervolgens enkel de aanvragen met kans boven een bepaalde treshold te behandelen waardoor de werklast drastisch verlaagt (met weerhoud van mogelijke ethische kwesties van het gebruik van AI op internationale bescherming).
Door ontbreken van deze data heb ik gebruik gemaakt van een dummy-data-set.
